# drone

서라벌고등학교 AI 파이오니어스 AI드론코딩 수업용

## 요구 환경

- Windows 10 이상
- Python 2.7.x

## 실행

```bash
$ pip install git+https://github.com/mcarral/python-qt5.git
$ python main.py
```
